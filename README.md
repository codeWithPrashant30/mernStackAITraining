# MERN Stack AI Training

This repository contains code for the MERN Stack AI Training course.

## Day 1: HTML Basics

The HTML folder contains examples of basic HTML elements and structure covered in Day 1:

- HTML document structure
- Headings, paragraphs, and text formatting
- Links and images
- Lists (ordered, unordered, and description lists)
- Semantic HTML elements
- Forms and input types
- Basic attributes
